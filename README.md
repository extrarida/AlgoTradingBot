# AlgoTrader Bot — FinTech Algorithmic Trading Bot
A Python-based algorithmic trading bot built with MetaTrader 5 API, Streamlit, and SQLite. <br>
Built for the CST4160 FinTech Coursework.

### 📋 Features

🔗 <b>MT5 Integration</b> — connects to MetaTrader 5 via the official Python API <br>
📊 <b>Live Price Feed</b> — real-time bid/ask prices for Forex and commodities <br>
📐 <b>Technical Indicators</b> — RSI, Moving Average, MACD <br>
🧠 <b>Automated Strategy</b> — combined signal from multiple indicators <br>
⚙️ <b>Order Execution</b> — automated BUY/SELL orders with stop-loss and take-profit <br>
🗄️ <b>SQLite Database</b> — trade history, price data, and system logs <br>
📈 <b>Performance Dashboard</b> — win rate, P&L charts, trade history <br>
🟢 <b>Demo Mode</b> — runs fully without a real MT5 account (simulated data) <br>
